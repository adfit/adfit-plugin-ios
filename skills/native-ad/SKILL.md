## Overview
AdFit SDK를 사용하여 네이티브 광고를 연동하려는 외부 개발자를 위한 스킬입니다.
네이티브 광고는 앱의 고유 UI에 맞춰 광고 애셋을 표시하는 형태이므로, 광고 뷰를 서비스에서 직접 구현해야 합니다.
이 스킬은 광고 로더 설정, 커스텀 광고 뷰 구현, 데이터 바인딩까지의 전체 연동 코드를 생성합니다.

## 실행 절차

1. 사용자에게 아래 정보를 확인합니다. (인자로 전달된 경우 생략)
   - **clientId**: 발급받은 광고 단위 ID

2. 확인된 정보를 바탕으로 [guide.md](guide.md)를 참고하여 연동 코드를 생성합니다.

3. 생성된 코드에 대해 간략한 설명을 제공합니다.
   - 네이티브 광고 뷰 구성 요소 (제목, 미디어, CTA 버튼 등)
   - 광고 로더 및 delegate 동작 방식
   - 미디어 비율 최적화 방법
